5. 운영 노트 / 알려진 한계

항목내용
서버 재시작 필요실행 중인 서비스는 이전 코드가 메모리에 있음 — stop.bat → run.bat 후 새 기능 활성화. 기존 DB는 MIGRATIONS가 자동 보강.
실 임베딩 키임베딩은 OpenAI 키 필요(text-embedding-3-small, ~$0.02/1M tok). 현재 키가 크레딧 소진 상태라 운영에선 exact-match로 자동 폴백(무해). 키 충전 또는 OPENAI_EMBED_MODEL+Azure 검토.
임베딩 인덱스 스코프유사도 인덱스는 레플리카 로컬(값은 Redis 공유 가능) — 멀티 레플리카에선 유사도 히트율만 소폭 하락, 정합성 문제 없음. 운영 확장 시 Redis 벡터 인덱스 검토.
복잡도 라우팅 기본값안전을 위해 전 에이전트 기본 OFF — 에이전트 정책 화면에서 토글 또는 자동 권고로 적용.
웹훅.env에 METIS_ALERT_WEBHOOK=https://hooks.slack.com/... 설정 시 활성. 미설정이면 인앱 알림만.
가드레일 임계METIS_GUARD_DROP_PCT(기본 10%), METIS_AUTO_REVERT(기본 1). 자동 원복은 구성변경+critical 알림으로 감사 추적됨.
