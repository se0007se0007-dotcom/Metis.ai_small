품질 게이트 동작 방식

강등(다운시프트) 후보가 발생하면 대상 모델의 최근 7일 품질 실적을 검사합니다.
승인 샘플 충분 + 평균 품질 ≥ 기준 → 강등 적용 · 카나리 샘플 부족 → 설정 비율만 강등하며 품질 데이터 수집 · 보류 품질 기준 미달 → 강등하지 않고 원 모델 유지 + 알림
복잡도 라우팅: 단순 프롬프트(복잡도 ≤0.3)가 상위 모델을 요청하면 경제 모델로 자동 강등 후보화 — 품질 게이트·거버넌스 방어를 통과해야 적용됩니다. 강등 run 품질이 10% 이상 떨어지면 가드레일이 자동 원복합니다.
