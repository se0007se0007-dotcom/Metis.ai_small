🎯 01. System Overview

Metis.AI는 엔터프라이즈 환경에서 AI Agent 운영을 위한 통합 거버넌스 SaaS 플랫폼입니다. 자연어 기반 워크플로우 빌더, 다중 에이전트 협업 커널, FinOps 비용 최적화, 릴리스 엔지니어링, 이상 탐지(FDS) 등 AI 운영에 필요한 전 주기를 하나의 플랫폼에서 관리합니다.

💰 10. FinOps Token Optimizer

3-Gate 아키텍처로 AI API 호출 비용을 최적화합니다. 각 게이트를 순차적으로 통과하며, 조기 반환이 가능합니다.

┌──────────────────┐ Request ────► │ Gate 1: Cache │ ── Hit ──► Return cached result │ (Semantic Match) │ └──────┬───────────┘ │ Miss ▼ ┌──────────────────┐ │ Gate 2: Router │ ── Route to cheapest capable model │ (Tier Selection) │ Tier-1 (Haiku/Flash) $0.25/M └──────┬───────────┘ Tier-2 (Sonnet/4o) $3/M │ Tier-3 (Opus/O3) $15/M ▼ ┌──────────────────┐ │ Gate 3: Packer │ ── Compress prompt + context │ (Skill Compress) │ into reusable skills └──────────────────┘

Model Tier Configuration

Tier Models Cost Use Case
Tier 1 Claude Haiku, Gemini Flash, GPT-4o Mini ~$0.25/M tokens Simple queries, classification, formatting
Tier 2 Claude Sonnet, GPT-4o, Gemini Pro ~$3/M tokens Complex analysis, multi-step reasoning
Tier 3 Claude Opus, O3, GPT-5 ~$15/M tokens Critical decisions, expert-level tasks

Tracking Metrics

모든 토큰 사용은 FinOpsTokenLog에 기록됩니다: cache hit 여부, routed tier, 원가 vs 최적화 비용, 응답 지연 시간 등을 추적하여 비용 절감 효과를 정량화합니다.

🚀 11. Release Engineering

AI 모델/워크플로우 버전 변경을 안전하게 배포하기 위한 3단계 검증 체계입니다.

🔁
Replay Testing
과거 실행 데이터셋을 새 버전에 재실행하여 회귀 검증
PASS FAIL REGRESSION IMPROVEMENT
👁
Shadow Traffic
프로덕션 트래픽을 후보 버전에 병렬 전송 (side-effect 없음)
Control vs. Candidate 결과 비교 분석
🐦
Canary Deployment
5% → 25% → 50% → 100% 단계적 트래픽 이동
Gate 평가: 성공률, 지연, 비용, 정책 위반

Promotion Decision Flow

Replay Pass → Shadow OK → Canary Gates Pass → PROMOTE
Any Gate Fail → ROLLBACK

Audit Log Schema

model AuditLog {
  id            String   @id @default(uuid())
  tenantId      String   // Multi-tenant isolation
  actorUserId   String   // Who performed the action
  action        String   // CREATE, UPDATE, DELETE, EXECUTE, PUBLISH...
  targetType    String   // Workflow, Pack, Execution, Agent...
  targetId      String   // Target entity ID
  policyResult  String?  // PASS / FAIL / WARN
  correlationId String   // End-to-end tracing
  metadataJson  Json?    // Additional context
  createdAt     DateTime @default(now())
}
